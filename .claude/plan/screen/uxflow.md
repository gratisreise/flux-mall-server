# UX Flow - flux-mall

## 1. 회원/인증

### 흐름
```
[로그인] → [회원가입] → [로그인]
    ↓
[메인 홈]
```

### 화면 구성 요소

#### 로그인
- 이메일 입력
- 비밀번호 입력
- 로그인 버튼
- 소셜 로그인 (Google, Kakao, Naver)
- 회원가입 링크

#### 회원가입
- 이메일 입력
- 비밀번호 입력
- 비밀번호 확인
- 이름 입력
- 휴대폰 번호 입력
- 약관 동의 (전체 동의, 이용약관, 개인정보처리방침)
- 가입 완료 버튼

---

## 2. 홈/탐색

### 흐름
```
[메인 홈] → [카테고리 목록] → [상품 리스트]
    ↓           ↓
    └──→ [검색 결과] → [상품 상세]
```

### 화면 구성 요소

#### 메인 홈
- 검색바
- 카테고리 메뉴
- 배너 슬라이드
- 인기 상품 섹션
- 신상품 섹션
- 할인 상품 섹션

#### 카테고리 목록
- 1차 카테고리 리스트
- 2차 카테고리 (선택 시)

#### 검색 결과
- 검색어
- 정렬 옵션 (정확도, 낮은가격, 높은가격, 최신순)
- 필터 (가격대, 브랜드, 배송유형)
- 상품 리스트 (썸네일, 상품명, 가격, 리뷰수)

---

## 3. 상품

### 흐름
```
[상품 리스트] → [상품 상세] → [장바구니 담기]
     ↓                            ↓
[상품 상세] → [상품 리뷰 상세]    [장바구니]
               ↓
           [상품 상세] ← 돌아가기
```

### 화면 구성 요소

#### 상품 리스트
- 썸네일 이미지
- 상품명
- 가격
- 할인율 (있는 경우)
- 리뷰 개수
- 장바구니 담기 버튼

#### 상품 상세
- 상품 이미지 (썸네일 + 확대)
- 상품명
- 가격
- 할인율
- 배송비
- 옵션 선택 (사이즈, 색상 등)
- 수량 선택
- 장바구니 담기 버튼
- 바로구매 버튼
- 찜하기 버튼
- 상품 설명
- 상품 리뷰 섹션

#### 상품 리뷰 상세
- 별점 평균
- 리뷰 개수
- 리뷰 필터 (별점, 사진리뷰)
- 리뷰 리스트 (작성자, 별점, 내용, 이미지, 날짜)

---

## 4. 장바구니/주문

### 흐름
```
[장바구니] → [주문서 작성] → [주문 완료] → [주문 내역 조회]
    ↓
[찜 목록] → [상품 상세] → [장바구니]
```

### 화면 구성 요소

#### 장바구니
- 선택 상품 체크박스
- 상품 썸네일, 이름, 옵션
- 수량 수정
- 개별 가격
- 삭제 버튼
- 전체 선택 / 전체 삭제
- 총 결제 금액
- 주문하기 버튼

#### 주문서 작성
- 주문 상품 정보
- 배송지 선택/입력 (기본 배송지, 신규 배송지)
- 수령인 정보
- 배송 요청사항
- 결제 수단 선택
- 상품 금액, 배송비, 할인금액, 총 결제금액
- 결제하기 버튼

#### 주문 완료
- 주문 번호
- 주문 일시
- 결제 금액
- 주문 상품 요약
- 주문 상세 조회 버튼
- 쇼핑 계속하기 버튼

#### 주문 내역 조회
- 주문 리스트 (주문번호, 날짜, 상품, 상태, 금액)
- 주문 상태별 필터 (전체, 결제완료, 배송중, 배송완료, 취소/반품)
- 주문 상세 버튼

#### 찜 목록
- 찜한 상품 리스트 (썸네일, 이름, 가격)
- 장바구니 담기 버튼
- 삭제 버튼

## 5. 마이페이지

### 흐름
```
[마이페이지 홈]
    ├── [주문/배송 조회] → [상품 상세]
    ├── [취소/반품/교환]
    ├── [찜 목록] → [상품 상세]
    ├── [배송지 관리]
    └── [회원 정보 수정]
```

### 화면 구성 요소

#### 마이페이지 홈
- 회원 정보 요약
- 최근 주문 내역
- 진행 중인 주문 (결제완료, 배송중)
- 빠른 메뉴 (주문조회, 취소/반품, 찜, 배송지)

#### 주문/배송 조회
- 주문 리스트
- 주문 상태 (결제완료, 상품준비중, 배송중, 배송완료)
- 배송조회 버튼
- 취소/반품 신청 버튼

#### 취소/반품/교환
- 주문 내역
- 취소/반품/교환 가능 상품
- 사유 선택
- 신청 버튼
- 신청 내역 조회

#### 찜 목록
- 찜한 상품 리스트 (썸네일, 이름, 가격)
- 장바구니 담기 버튼
- 삭제 버튼

#### 배송지 관리
- 배송지 리스트 (수령인, 주소, 연락처)
- 기본 배송지 설정
- 배송지 추가/수정/삭제

#### 회원 정보 수정
- 이름
- 휴대폰 번호
- 이메일 (수정 불가)
- 비밀번호 변경

---

## 6. 판매자

### 흐름
```
[판매자 홈]
    ├── [상품 등록]
    ├── [상품 관리] → [상품 수정/삭제]
    ├── [주문 관리]
    └── [판매 내역 조회]
```

### 화면 구성 요소

#### 판매자 홈
- 판매자 정보 요약
- 오늘/이번달 판매 현황
- 최근 주문 내역
- 빠른 메뉴 (상품등록, 상품관리, 주문관리)

#### 상품 등록
- 상품명
- 카테고리 선택
- 상품 이미지 등록
- 가격
- 할인율
- 재고 수량
- 옵션 (사이즈, 색상 등)
- 상품 설명

#### 상품 관리
- 상품 리스트 (썸네일, 이름, 가격, 재고, 승인상태)
- 수정 버튼
- 삭제 버튼
- 품절/판매중지 토글

#### 주문 관리
- 주문 리스트 (주문번호, 상품, 수량, 주소, 상태)
- 배송시작 버튼
- 송장번호 입력
- 배송완료 처리

#### 판매 내역 조회
- 판매 날짜별/월별 통계
- 판매 금액 합계
- 주문 내역 상세

---

## 7. 관리자

### 흐름
```
[관리자 홈]
    ├── [회원 관리]
    ├── [판매자 관리]
    ├── [상품 승인/관리]
    ├── [주문/배송 전체 관리]
    └── [카테고리 관리]
```

### 화면 구성 요소

#### 관리자 홈 (대시보드)
- 전체 회원/판매자/상품/주문 수
- 오늘 가입/판매 현황
- 승인 대기 상품 수
- 최신 주문/회원가입 내역

#### 회원 관리
- 회원 리스트 (이름, 이메일, 가입일, 상태)
- 검색/필터
- 강제 탈퇴
- 상태 변경 (활성/비활성)

#### 판매자 관리
- 판매자 리스트 (상호명, 대표자, 연락처, 승인상태)
- 승인/거절
- 판매자 상세 정보

#### 상품 승인/관리
- 승인 대기 상품 리스트
- 상품 상세
- 승인/거절
- 전체 상품 관리 (수정, 삭제, 강제 삭제)

#### 주문/배송 전체 관리
- 전체 주문 리스트
- 주문별 상태 확인
- 주문 취소 처리

#### 카테고리 관리
- 1차/2차 카테고리 리스트
- 카테고리 추가/수정/삭제

---

## 전체 흐름 요약

```
비회원: 메인홈 → 상품탐색 → 로그인/회원가입
회원: 메인홈 → 상품탐색 → 장바구니 → 주문 → 마이페이지
판매자: 판매자홈 → 상품관리 → 주문관리 → 판매내역
관리자: 관리자홈 → 회원/판매자/상품/주문 관리
```
