### 1. 회원 (Member) 도메인

- 회원가입: 이메일/비밀번호 기반 계정 생성 (아이디·닉네임 중복 체크 포함)
- 로그인/로그아웃: JWT기반 인증 토큰 발급 및 만료 처리
- 내 정보 관리: 프로필 정보(닉네임 등) 조회 및 수정
- 배송지 관리: 기본 배송지 설정 및 다중 배송지 CRUD (등록·수정·삭제·조회)
- 권한 관리: 일반 사용자 / 관리자 역할 구분 및 관리자 페이지 접근 제어 (RBAC)

### 2. 상품 (Product) 도메인

- 상품 목록 조회: 페이징 처리, 카테고리·가격대·정렬 옵션 필터링
- 상품 키워드 검색: 상품명·설명 기반 검색 (부분 일치)
- 상품 상세 조회: 단일 상품 상세 정보, 실시간 재고량, 평균 평점 노출
- 상품 관리 (SELLER): 신규 상품 등록, 정보 수정, 삭제 (Soft Delete)
- 품절 처리: 재고 0 시 자동으로 품절 상태 변경 및 표시

### 3. 장바구니 (Cart) 도메인

- 아이템 추가: 특정 상품을 수량 지정하여 장바구니에 담기 (재고 초과 방지)
- 아이템 수정: 담긴 상품 수량 증감(재고 초과 방지)
- 아이템 삭제: 개별 삭제 및 선택 상품 일괄 삭제
- 장바구니 조회: 현재 담긴 상품 리스트, 예상 총 결제 금액 실시간 계산
- 재고 유효성 체크: 장바구니 조회 및 주문 진행 시 실제 재고와 비교

### 4. 주문 (Order) 도메인

- 주문서 생성: 장바구니 전체 주문 또는 개별 상품 바로구매
- 결제 요청/검증: 실제 결제 연결없음 모든 결제는 일다 성공하는 걸로
- 재고 차감: 결제 승인 완료 시 실시간 재고 수량 감소 (동시성 제어)
- 주문 내역 조회: 회원별 주문 목록 (페이징), 주문 상세 현황 조회
- 주문 취소/환불: 전체 취소 또는 부분 취소 처리, 재고 원복 및 결제 취소 연동

### 5. 위시리스트 (Wishlist) 도메인

- 상품 찜하기: 상품 목록·상세 페이지에서 찜 추가/제거 (하트 토글)
- 위시리스트 조회: 찜한 상품 리스트 전체 보기 (페이징, 정렬)
- 재입고 알림 신청: 품절 상품에 대해 재입고 시 알림 설정 (이메일 또는 푸시)
- 장바구니 연동: 위시리스트 상품을 장바구니로 일괄 또는 개별 이동

### 6. 최근 본 상품 (Recently Viewed) 도메인

- 자동 기록: 상품 상세 페이지 방문 시 자동 저장 (로그인 회원은 DB, 비로그인 시 로컬스토리지)
- 최근 본 상품 노출: 홈페이지, 상품 상세 페이지 하단 등에 리스트 형태 표시
- 중복 제거 및 제한: 최근 20~50개 상품만 유지 (오래된 순서대로 자동 삭제) ⇒ 배치??, 스케줄링??

### 7. 상품 리뷰 & 평점 (Review) 도메인

- 리뷰 작성: 구매 확정 후 텍스트, 사진 업로드, 별점(1~5점) 등록
- 리뷰 조회: 상품 상세 페이지에 리뷰 목록 표시, 평균 평점 계산 및 노출
- 리뷰 필터링/정렬: 사진 있는 리뷰 우선, 최신순·평점 높은 순·도움이 돼요 순 정렬
- 베스트 리뷰 관리: 도움이 돼요 투표 기능, 상위 리뷰 우선 노출
- 리뷰 관리 (Admin): 부적절 리뷰 신고 처리 및 삭제
